# 西行闯关 · 九九八十一难

一个以《西游记》为主题的网页问答闯关游戏。

## 快速开始

用浏览器打开 `index.html` 即可开始游戏，无需安装任何依赖。

## 玩法

- 从长安出发，一路答西游记知识题向西推进，到达大雷音寺即通关
- 共 **12关**，包含多种关卡类型：普通问答、BOSS战、宝物关、迷宫、渡劫关、终极考验
- **50道**西游记知识题库，每局随机抽取
- **6件法宝**助阵：锦斓袈裟、紧箍咒、筋斗云、火眼金睛、人参果、金箍棒
- 初始 **5条命**，答错扣命，命用完游戏结束

## 快捷键

| 键 | 功能 |
|----|------|
| Q~Y | 使用法宝 |
| 1~4 | 选择答案 |
| 空格 | 继续下一题 |

## 技术

纯前端 HTML/CSS/JS，单文件，无外部依赖。
