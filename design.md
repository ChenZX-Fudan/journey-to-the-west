# 西游记 · 西行闯关 — 游戏设计文档

## 一、游戏概述

**类型**：网页问答闯关 + 棋盘冒险  
**技术**：纯前端 HTML/CSS/JS，单文件，无需外部依赖  
**目标体验**：边答西游记知识题边推进西行地图，法宝系统增加策略性

---

## 二、核心玩法循环

```
 ┌──────────┐    答对     ┌──────────┐
 │ 站在当前  │ ─────────→ │ 前进1格   │ → 触发格子事件
 │ 棋盘格上  │            └──────────┘
 └──────────┘
      │ 答错
      ▼
 ┌──────────┐
 │ 扣1条命   │ → 留在原地，重新出题
 └──────────┘
```

- 从长安出发，一路向西到雷音寺取经
- 共 **12 个关卡节点**（对应12段经典剧情）
- 走到终点 = 取经成功 = 通关

## 四、法宝系统

玩家初始拥有 **5条命（紧箍环）**，沿途可获取法宝：

| 法宝 | 图标 | 效果 | 获取方式 |
|------|------|------|----------|
| 锦斓袈裟 | 👘 | 跳过当前题目（不前进也不扣命） | 答对3题连续奖励 |
| 紧箍咒 | 🔔 | 排除2个错误选项（50/50） | 初始自带2个 |
| 筋斗云 | ☁️ | 直接前进2格（跳过中间题目） | 宝物关获得 |
| 火眼金睛 | 👁️ | 显示答案提示（文字线索） | 第5关后解锁 |
| 人参果 | 🍑 | 回复1条命 | 宝物关概率获得 |
| 金箍棒 | 🏏 | BOSS战中自动消除一次答错惩罚 | 初始自带1个 |

## 六、关卡类型详解

### 普通问答关（第1-5关）
- 出现一道题目，4选1
- 倒计时 30 秒
- 答对前进，答错扣命 + 重新出题
- 连续答对3题奖励一个法宝

### BOSS战（第6、8、11关）
- 需要**连续答对 N 题**（N=3/3/4）
- 期间答错一次可以用金箍棒抵消，否则从头开始
- BOSS有专属血条和动画
- 通过后有专属"收妖"动画

### 宝物关（第7关）
- 答对题目即可随机获得一件法宝
- 趣味：人参果树，摇一摇掉宝贝

### 迷宫关（第9关）
- 模拟盘丝洞迷宫
- 3格迷宫：答对前进、答错后退
- 走出迷宫 = 过关

### 渡劫关（第10关）
- 老鼋（通天河大龟）问卦
- 只出一题，答对直接过关，答错扣2命

### 终极大关（第12关）
- 5题限时连答（每题20秒）
- 可使用全部法宝
- 通关 = 取经成功 = 胜利画面

## 九、技术实现要点

- **单 HTML 文件**，CSS + JS 均内嵌
- **横向棋盘**使用 CSS `overflow-x: auto` 可拖拽滚动，自动跟随当前位置
- **题目数据**用 JS 数组存储，随机抽取不重复
- **状态管理**用简单的 JS 对象，记录当前位置、生命、法宝、分数
- **动画**用 CSS transition/animation，配合少量 JS 触发
- **响应式**适配桌面和手机（竖屏时地图改为纵向）
- **音效可选**：用简单的 Web Audio API 做反馈音效（点击、答对、答错）
